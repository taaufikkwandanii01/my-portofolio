Project

Aplikasi Monitoring Suporter Sepak Bola Ketika Bertandang

Merancang aplikasi android untuk membantu suporter sepak bola dalam melakukan monitoring perjalanan ketika bertandang.

  • Fitur utama: Berbagi lokasi, memberikan notifikasi informasi ketika terdapat suporter di dalam rombongan yang tertinggal, dan memberikan informasi kontak mengenai suporter yang tertinggal tersebut.
  • Tools: Android studio, Java, Firebase, Google Maps API, dan OneSignal.
  • Memberikan kemudahan bagi suporter dalam melakukan monitoring rombongan suporter nya, dengan presentase sebesar 95,71%
Sistem Skill Matriks Berbasis Web Untuk Admin

Merancang website untuk membantu admin dalam melakukan pengelolaan data skill matriks karyawan pada divisi aircraft.

  • Fitur utama: Pengelolaan data karyawan, monitoring masa kadaluarsa sertifikasi keterampilan karyawan pada divisi aricraft.
  • Tools: Visual studio code, PHP, JavaScript, MySql.
  • Memberikan kemudahan bagi admin dalam melakukan pengelolaan serta monitoring massa sertifikasi keterampilan karyawan, dengan presentase sebesar 100%
